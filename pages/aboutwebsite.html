About us

This website is a joint effort by 3 1st year students at Griffith College Cork, Ireland. It was created as a final group project for the Front-End Web Development project. We decided to create an authentic community website for a dog club and for that, a small quiz was conducted by Ilia to determine the potential product owner's needs.

The website structure is split in multiple thematically-independent pages tied together by an interactive menu, allowing for easy navigation. We tried to keep the look and structure minimalistic except for the text-heavy pages with sport history and information on the club.

After the rough requirements were outlined, the responsibilities were split in the following way:

  • Ilia - project structure, HTML, content and documentation
  • Alfred - Javascript and HTML pertaining to interactive elements
  • Saif - CSS

A git repository was used to hold the project data, allow easy collaboration, version control and concurrent and independent work on various website elements. The repository with the change history can be found here.
